# 08-skill-training SPEC

## 目标

实现首页“技能培训”入口对应的静态课程浏览页，让用户能快速看懂培训方向、地点、时间和费用说明。

## 页面范围

- 页面路由：`pages/training/index`
- 页面数据：`data/training.js` 与 `data/training.ts`
- 包含培训分类、推荐课程、开班信息和诚实的静态能力反馈。
- 视觉沿用现有小程序，优先大字、短句、清楚标签。

## 暂不实现

- 真实报名、支付、学习记录、证书和后端 API。
- 不修改首页、`app.json` 或其他模块。

## 验收

- 页面可独立渲染，mock 数据结构可替换为 API 返回。
- 主要信息无需专业术语即可理解。
