# 12-return-home SPEC

## 目标

实现首页“返乡专区”入口对应的返乡就业服务页，集中展示家乡岗位、创业支持和返乡服务提醒。

## 页面范围

- 页面路由：`pages/return-home/index`
- 页面数据：`data/return-home.js` 与 `data/return-home.ts`
- 包含返乡欢迎区、本地机会、服务政策和办理地点。
- 文案亲切但务实，避免宣传口号占据主要空间。

## 暂不实现

- 创业申请、资格审核、地图导航和后端 API。
- 不修改首页、`app.json` 或其他模块。

## 验收

- 页面能让用户快速找到“工作、创业、服务”三类信息。
- mock 数据结构适合未来 API 接入。
