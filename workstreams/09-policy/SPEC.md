# 09-policy SPEC

## 目标

实现首页“看政策”入口对应的政策信息页，用通俗语言展示就业补贴、培训支持和办事提醒。

## 页面范围

- 页面路由：`pages/policy/index`
- 页面数据：`data/policy.js` 与 `data/policy.ts`
- 包含政策分类、重点提醒、政策列表、发布日期和适用人群。
- 避免长篇公文堆叠，先给用户“谁能办、有什么用、去哪里办”。

## 暂不实现

- 在线申报、身份核验、政务接口和后端 API。
- 不修改首页、`app.json` 或其他模块。

## 验收

- 页面结构清楚，政策内容使用短句和清晰标签。
- 数据结构可在后续直接替换为 API 数据。
