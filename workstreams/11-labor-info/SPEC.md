# 11-labor-info SPEC

## 目标

实现首页“用工信息”入口对应的信息页，展示企业近期用工需求、短期用工和紧急招工提醒。

## 页面范围

- 页面路由：`pages/labor/index`
- 页面数据：`data/labor.js` 与 `data/labor.ts`
- 包含用工类型筛选、需求列表、人数、工期、地点和联系电话提示。
- 信息表达直接，适合快速浏览和现场服务场景。

## 暂不实现

- 电话真实拨打、在线报名、企业审核和后端 API。
- 不修改首页、`app.json` 或其他模块。

## 验收

- 页面可独立渲染，重点信息醒目但不拥挤。
- mock 数据可平滑替换为 API 数据。
