# 10-campus SPEC

## 目标

实现首页“校园招”入口对应的校园就业页，服务学生和应届毕业生查看校招岗位、双选会和实习机会。

## 页面范围

- 页面路由：`pages/campus/index`
- 页面数据：`data/campus.js` 与 `data/campus.ts`
- 包含校招活动、应届岗位、实习标签和时间地点。
- 采用与现有首页一致的清晰、务实视觉风格。

## 暂不实现

- 简历投递、学校认证、企业报名和后端 API。
- 不修改首页、`app.json` 或其他模块。

## 验收

- 页面文件完整，核心信息在首屏易于扫描。
- mock 数据字段能支持后续 API 对接。
