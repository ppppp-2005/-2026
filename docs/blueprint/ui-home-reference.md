# 首页视觉参考

## 已确认方向

首页以 ImageGen 生成并被确认喜欢的方案为视觉目标：

- 顶部家乡形象区域
- 圆形 logo 占位
- 标题：家乡就业
- 副标题：找工作 招工人 学技能
- 登录入口
- 搜索框
- 8 个功能入口
- 最新资讯
- 热门服务
- 底部 Tab：首页 / 职位 / 消息 / 我的

## 首页区域顺序

1. 顶部品牌与形象区
2. 搜索框
3. 功能宫格
4. 最新资讯
5. 热门服务
6. 底部 Tab

## 建议文案

- 搜索框：输入工种 / 地点 / 企业
- 功能入口：找工作、企业招人、技能培训、看政策、校园招、用工信息、返乡专区、培训报名
- 热门服务：招聘会、最新岗位、企业专区、我要招人

## 视觉规则

- 不复制原参考图里的政府 logo、单位名称或真实图片。
- 当前没有正式素材时，用简单色块占位。
- 文字必须在代码里真实写入，不依赖生成图片里的文字。
- 首页风格要像地方就业服务小程序，不要做成高端互联网招聘 App。
