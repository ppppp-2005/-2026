社会服务赛道 · 家乡就业服务

让工作机会，离家乡群众更近一点

“三都职通”是一款服务三都县群众与本地用人单位的招聘小程序创意， 让群众少跑腿、让企业少下乡摆摊，让就业信息更快、更清楚地到达真正需要的人。

创意名称

三都职通

  • 群众在家也能看到合适岗位
  • 企业用更低成本触达求职者
  • 政府与劳务服务人员更容易组织就业服务

01 · 项目缘起

来自家人的真实工作，也来自家乡的真实困难

家人长期从事劳务输出，日常需要组织群众到外省务工。这个项目不是凭空想象， 而是从一次次线下找人、打电话、贴公告和下乡招工中长出来的。

群众这一边

有劳动能力，却不知道哪里正在招人

部分群众受教育程度不高、数字渠道有限，找工作常依赖熟人介绍或前往人力资源公司询问。 想找本地工作或外出务工，都需要花费较多时间和路费。

用人单位这一边

有岗位需求，却很难高效找到合适的人

本地商户仍会把招聘信息贴在公告栏，企业有时需要从县城到乡镇摆摊招聘。 信息传播范围有限，招聘成本高，供需双方常常彼此错过。

创作契机： 在学习编程和接触 Vibe Coding 后，我意识到智能体可以帮助普通学生把真实需求变成可验证的产品原型。 我希望借助比赛，先把家乡就业服务这件事认真做出第一步。

02 · 目标用户

不只服务一种人，而是连接整个县域就业供需

产品优先服务真实、常见、容易被忽视的就业场景，不把功能做得过度复杂， 让第一次使用智能手机求职的人也能看懂、敢点、知道下一步。

USER 01

普通务工群众

寻找县内服务业、制造业或外省务工岗位。

USER 02

持证技术工人

按叉车、电工、焊工等技能快速匹配岗位。

USER 03

本地商户与企业

发布真实岗位，减少反复下乡招人的成本。

USER 04

就业服务人员

集中维护岗位、活动、培训和政策信息。

    04 · 产品方案

    先把最有价值的就业服务做好

    产品不追求一次覆盖所有政务服务，第一阶段专注“看岗位、懂岗位、能投递、有人处理、收到结果”这条核心链路。

    群众端

    职位搜索与清楚筛选

    按地区、工种、薪资和技能查看岗位，文案简短，大字展示，减少理解成本。

    群众端

    简历与投递进度

    维护基本经历和技能，投递后清楚看到待查看、面试或结束状态。

    企业端

    岗位发布与候选人处理

    企业经过审核后发布岗位，集中查看求职者并更新处理结果。

    就业服务

    招聘会与宣讲活动

    集中展示招聘会、直播带岗和宣讲会，让线下活动更容易被群众看见。

    就业服务

    培训与政策信息

    提供技能培训、就业政策、劳动用工和返乡服务入口，帮助群众做好求职准备。

    适老易读

    简单、诚实、容易操作

    按钮足够大，重要状态说清楚；未接入的能力不提示虚假成功，避免误导群众。

    当前已经完成的原型

    • 首页、职位、交流、消息、个人中心五个核心入口
    • 职位详情、本地筛选、简历、投递记录和企业招聘框架
    • 招聘活动、培训、政策、返乡与用工信息页面
    • 加载、空状态、异常和重试等基础体验

    真正上线前必须补齐

    • 真实微信登录、手机号和角色权限
    • 后端、数据库、企业认证和岗位审核
    • 真实投递、状态通知、文件上传和数据保护
    • 微信开发者工具、真机和不同字体尺寸验收

    05 · 价值与意义

    技术的价值，不是功能多，而是让真实的人少走弯路

    三都职通希望成为县域就业服务的一块数字基础设施：降低信息差，让岗位、人才和公共就业服务在同一个入口相遇。

    少跑腿

    降低群众求职成本

    在家即可了解岗位、活动和培训信息，把时间和路费留给真正需要的面试与上岗。

    快触达

    降低企业招聘成本

    减少反复张贴公告和下乡摆摊，让岗位更精准地被合适的求职者看见。

    可持续

    沉淀本地就业服务能力

    逐步形成可信岗位、技能人才和就业活动数据，为后续公共服务协作提供基础。

    06 · 落地计划

    从可展示原型，走向真正有人使用

    不盲目追求大而全。先用一个县、一个核心就业闭环验证价值，再根据真实使用反馈扩展培训、返乡和公共服务。

    阶段 01 · 当前

    完成前端原型

    验证页面结构、操作方式和群众能否看懂。

    阶段 02 · MVP

    打通真实求职闭环

    完成登录、岗位、投递、企业处理和通知。

    阶段 03 · 试点

    邀请真实用户试用

    与家人、本地商户和少量求职群众开展小范围测试。

    阶段 04 · 协作

    连接公共就业服务

    在数据合规前提下，与企业及相关服务单位逐步合作。

    从一条真实招聘信息开始，让更多家乡群众看到工作的可能

    三都职通不是要取代线下就业服务，而是让线下经验拥有一个更高效、更可信、更容易触达群众的数字入口。
